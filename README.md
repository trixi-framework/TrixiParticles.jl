TrixiParticles.jl is a code that implements particle based numerical methods for multiphysics applications.

## Features
- Incompressible Navier-Stokes
  - Methods: WCSPH, EDAC
- Solid-body mechanics
  - Methods: TLSPH
- Fluid-Structure Interaction
- Output formats:
  - VTK
